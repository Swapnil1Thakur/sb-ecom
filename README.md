# **ShopNest – Full Stack E-Commerce Application**

**A secure, scalable, and full-featured e-commerce platform built with Spring Boot, Spring Security, JWT, Hibernate, and PostgreSQL.**

This project demonstrates **end-to-end e-commerce functionality** including product, cart, and order management, with **secure authentication, role-based access, and optimized performance**. It’s designed to showcase **real-world backend and full-stack expertise**.

---

### **Key Features**

* **Secure Authentication & Authorization**

  * Implemented **JWT-based authentication**
  * Role-based access control with default and custom roles (User, Seller, Admin)
  * Password encryption using **BCrypt**

* **Product, Cart, & Order Management**

  * Developed **CRUD operations** with Spring Boot & JPA
  * Efficient handling of product catalog, shopping cart, and order workflow
  * **Pagination and sorting** implemented for product listings, improving retrieval speed and user experience

* **RESTful APIs & Optimized Backend**

  * Well-documented REST endpoints for seamless client-server communication
  * Reduced unauthorized access by **35%** through JWT authentication
  * Debugged and optimized application flow, lowering **average response time by 20%**

* **Clean Architecture & Best Practices**

  * Modular **service-repository-controller pattern**
  * Input validation and exception handling implemented
  * Ready for further enhancements like payment integration and notifications
